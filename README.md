Overview

The Rental Property Management System is an application and website designed to help manage rental properties efficiently. The system allows property managers or landlords to register tenants, manage volunteers or staff, and view or update tenant information in one centralized platform.

The project includes:

A management application

A PHP-based website

Database integration for storing tenant and property information

This system simplifies property administration and record management.

Features

Register new tenants

Manage tenant records

Add and manage volunteers or staff

View tenant information

Update tenant details

User-friendly management interface

Web-based access using PHP backend

HTML5

CSS3

JavaScript

MySQL Database

Management application (Java / Desktop app if applicable)
